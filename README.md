# Browser.js

> A sugar for your browser.

## API

### visit( url )

Type: `String`

Redirect to the url given.

```
Brower.visit('http://example.com')
return false
```

### preventEmbedded()

Redirect to the right page if embedded.

```
Brower.preventEmbedded()
```

### isEmbedded()

Tell you if the page is embedded (iframe) or not.

```
Brower.isEmbedded()
```
